Chapter 6: Additional Aspects of Aqueous Equilibria üßÆ

Applying equilibrium concepts to complex systems like buffers, titrations, and solubility.

Core Concepts

Common-Ion Effect

The suppression of dissociation of a weak electrolyte by adding a common ion. Adding NaF to a solution of HF shifts the equilibrium HF ⇌ H+ + F- to the LEFT.

Buffer Solutions

  • What they are: A solution that resists changes in pH.
  • Composition: A weak acid + its conjugate base (e.g., HF + NaF) OR a weak base + its conjugate acid (e.g., NH3 + NH4Cl).
  • How they work: The conjugate base (A-) neutralizes added H+. The weak acid (HA) neutralizes added OH-.

Titrations

A procedure to determine the concentration of an unknown by reacting it with a known solution.

  • Equivalence Point: Moles of titrant = Moles of analyte.
  • Half-Equivalence Point: (For weak titrations) Half of the analyte has been neutralized. At this point, pH = pKa.
  • pH at Equivalence Point:
    • Strong Acid/Strong Base: pH = 7.00
    • Weak Acid/Strong Base: pH > 7.00 (conjugate base is present)
    • Weak Base/Strong Acid: pH < 7.00 (conjugate acid is present)

Solubility Equilibria (Ksp)

The equilibrium for "insoluble" salts. Molar Solubility (s) is the moles of salt that dissolve in 1 L.

  • AgCl(s) ⇌ Ag+(aq) + Cl-(aq)
  • Ksp = [Ag+][Cl-]
  • In this case, Ksp = (s)(s) = s2
  • PbI2(s) ⇌ Pb2+(aq) + 2I-(aq)
  • Ksp = [Pb2+][I-]2
  • In this case, Ksp = (s)(2s)2 = 4s3

Predicting Precipitation (Q vs Ksp):

  • Q < Ksp: Unsaturated. No precipitate.
  • Q > Ksp: Supersaturated. Precipitate will form.
  • Q = Ksp: Saturated. At equilibrium.
Key Skills & Problem Solving

Henderson-Hasselbalch Equation (Buffers)

A shortcut for buffer pH calculations. ONLY FOR BUFFERS.

pH = pKa + log([Base] / [Acid])

Titration pH Calculations (4 Regions)

  1. Initial pH: A Ch. 5 weak acid/base problem.
  2. Buffer Region (before eq. point): A buffer problem. Use stoichiometry ("before/after" table) then H-H equation.
  3. Equivalence Point: A Ch. 5 salt hydrolysis problem.
  4. Past Equivalence Point: A strong acid/base problem (determined by excess titrant).

Ksp and Molar Solubility (s)

Use an ICE table (omitting the solid) to relate Ksp and s. Remember the common-ion effect will decrease solubility.

Common Pitfalls
  • Misusing H-H equation. It only works for buffers, not at the start of a titration or at the equivalence point.
  • Confusing Molar Solubility (s, in mol/L) with Ksp (the constant).
  • Forgetting stoichiometry in the Ksp expression (e.g., (2s)2).
  • Assuming the equivalence point of a weak titration is at pH 7. It is not!
  • Forgetting to account for dilution when two solutions are mixed (for Q vs Ksp problems).
Expected Question Types
  1. Given: [Weak Acid] and [Conj. Base]. Find: pH.
    • Identify the solution as a buffer.
    • Use the Henderson-Hasselbalch (H-H) equation: pH = pKa + log([Base]/[Acid]).
    • If given Ka, find pKa = -log(Ka).
    • Plug in concentrations and solve for pH.
    • (Alternatively: Use a full ICE table, but H-H is faster).
  2. Given: A target pH. Find: Which acid/base pair is best to make a buffer?
    • A buffer works best when its pH ≈ pKa.
    • Calculate the pKa = -log(Ka) for each available acid.
    • Choose the acid whose pKa is closest to the target pH.
  3. Given: A buffer. Find: New pH after adding a small amount of strong acid/base.
    • Part 1: Stoichiometry (Before/After Table)
      • Work in moles (moles = M √ó L).
      • Write the reaction:
        • If adding H+: A- + H+ → HA
        • If adding OH-: HA + OH- → A- + H2O
      • Use a "Before/Add/After" table to find the new moles of HA and A-.
    • Part 2: Equilibrium (H-H Equation)
      • Use the new moles from the "After" row in the H-H equation.
      • pH = pKa + log(moles Base / moles Acid).
      • (You can use moles directly since the volume is the same and would cancel out).
  4. Given: A titration setup. Find: The pH at the 4 key regions.
    • 1. Initial pH (0 mL titrant):
      • This is a Chapter 5 weak acid/base problem. Use an ICE table and Ka (or Kb).
    • 2. Buffer Region (after adding titrant, before eq. point):
      • This is a buffer problem. Use a "Before/After" stoichiometry table to find moles of HA and A-, then use the H-H equation.
      • Shortcut (Half-Equivalence Point): At this specific point, pH = pKa.
    • 3. Equivalence Point:
      • All the initial acid/base has been converted to its conjugate.
      • This is a Chapter 5 salt hydrolysis problem.
      • Find the new total volume, calculate the concentration of the conjugate, and use Kb (or Ka) with an ICE table to find pOH (or pH).
    • 4. Past Equivalence Point:
      • The pH is now determined only by the excess strong titrant.
      • Find moles of excess H+ or OH-.
      • Find the new total volume.
      • Calculate [H+] or [OH-] = (moles excess) / (total L).
      • Calculate pH or pOH.
  5. Given: Molar solubility (s). Find: Ksp.
    • Write the dissociation equation (e.g., PbI2(s) ⇌ Pb2+ + 2I-).
    • Write the Ksp expression (e.g., Ksp = [Pb2+][I-]2).
    • Relate the ion concentrations to s based on stoichiometry:
      • [Pb2+] = s
      • [I-] = 2s
    • Substitute these s values into the Ksp expression: Ksp = (s)(2s)2 = 4s3.
    • Plug in the given value for s and solve for Ksp.
  6. Given: Ksp. Find: Molar solubility (s) in pure water.
    • This is the reverse of Question 5.
    • Write the dissociation and Ksp expression.
    • Set up a simple ICE table where s is the change.
      • Equilibrium concentrations will be s, 2s, etc.
    • Substitute these s expressions into the Ksp expression (e.g., Ksp = 4s3).
    • Plug in the given value for Ksp and solve for s.
  7. Given: Ksp. Find: Molar solubility (s) in a solution with a common ion.
    • Write the dissociation and Ksp expression.
      • Set up an ICE table.
    • The "I" (Initial) row will be zero for one ion, but will be the given concentration for the common ion (e.g., [I-]initial = 0.1 M).
    • The "C" (Change) row is +s, +2s, etc.
    • The "E" (Equilibrium) row will be s and 0.1 + 2s.
    • Substitute "E" values into Ksp: Ksp = (s)(0.1 + 2s)2.
    • Assume s is small (e.g., 0.1 + 2s ≈ 0.1).
    • Solve the simplified equation for s.
  8. Given: Two mixed solutions. Find: Will a precipitate form?
    • Identify the potential precipitate (the insoluble salt).
    • Write its Ksp expression. This is your target for Q.
    • Part 1: Dilution
      • Calculate the new total volume (Vfinal = V1 + V2).
      • Calculate the new, diluted concentrations of the relevant ions using M1V1 = M2V2.
    • Part 2: Calculate Q
      • Plug these new diluted concentrations into the Q expression.
    • Part 3: Compare Q vs. Ksp
      • If Q > Ksp: Yes, a precipitate will form.
      • If Q < Ksp: No, the solution is unsaturated.

Chapter 4: Chemical Equilibrium ⚛️

Understanding the balance between reactants and products in reversible reactions.

Core Concepts

Dynamic Equilibrium

A state where the rate of the forward reaction equals the rate of the reverse reaction. Concentrations of reactants and products remain constant, but the reactions have not stopped.

Equilibrium Constant (K)

For aA + bB ⇌ cC + dD:

  • Kc = [C]c[D]d / [A]a[B]b (using Molarity)
  • Kp = (PC)c(PD)d / (PA)a(PB)b (using Partial Pressure)
  • CRUCIAL: Pure solids (s) and liquids (l) are excluded.

Interpreting K

  • K > 105: Product-favored. Reaction goes to completion.
  • K < 10-5: Reactant-favored. Little product forms.
  • Intermediate K: Appreciable amounts of both.

Kp vs Kc

Kp = Kc(RT)Δn where Δn = (moles of gas products) - (moles of gas reactants).

Reaction Quotient (Q)

Calculated the same as K, but with non-equilibrium concentrations.

  • Q < K: Shifts RIGHT (→) to make more products.
  • Q > K: Shifts LEFT (←) to make more reactants.
  • Q = K: System is at equilibrium.

Le Châtelier's Principle

When a system at equilibrium is stressed, it shifts to relieve that stress.

  • Concentration: Add substance, shift away. Remove substance, shift toward.
  • Pressure (Gases): Increase P, shift to side with FEWER moles of gas.
  • Temperature: Increase T, shift in ENDOTHERMIC direction (treat heat as a reactant/product). This is the only stress that changes the value of K.
Key Skills & Problem Solving

Solving with ICE Tables

  1. I (Initial): Fill in initial concentrations.
  2. C (Change): Define change in terms of 'x', using stoichiometry.
  3. E (Equilibrium): Sum the I and C rows.
  4. Substitute "E" values into the K expression and solve for x.

5% Approximation: If K is very small, you can assume [Initial] - x ≈ [Initial]. Always check your assumption afterward: (x / [Initial]) * 100% < 5%.

Common Pitfalls
  • Forgetting to exclude solids (s) and liquids (l) from the K expression.
  • Forgetting to use coefficients as exponents in the K expression.
  • Messing up stoichiometry in the "C" row of an ICE table (e.g., writing -x instead of -2x).
  • Mixing up Q (snapshot) and K (equilibrium).
  • Forgetting to check the 5% approximation.
Expected Question Types
  1. Given: A reaction. Find: The Kc or Kp expression.
    • Write the expression as [Products] / [Reactants].
    • For Kc, use concentrations [ ]. For Kp, use partial pressures PX.
    • Raise each species to the power of its stoichiometric coefficient.
    • Crucially: Exclude any pure solids (s) or pure liquids (l).
  2. Given: Equilibrium concentrations. Find: The value of K.
    • Write the correct K expression (see steps above).
    • Substitute the given equilibrium concentrations or pressures directly into the expression.
    • Calculate the final value.
  3. Given: Initial concentrations and one equilibrium concentration. Find: The value of K.
    • Set up an ICE table (Initial, Change, Equilibrium).
    • Fill in the "I" row with the given initial concentrations.
    • Use the one known equilibrium concentration to solve for the "C" (Change) value, x.
    • Use x to calculate all other "E" (Equilibrium) concentrations based on stoichiometry.
    • Write the K expression.
    • Plug all the calculated "E" values into the K expression and solve for K.
  4. Given: K and initial concentrations. Find: Equilibrium concentrations.
    • Set up an ICE table.
    • Fill in the "I" row with initial concentrations.
    • Fill in the "C" row using +x and -x, applying stoichiometric coefficients (e.g., -2x, +3x).
    • Fill in the "E" row by combining the "I" and "C" rows (e.g., 0.10 - 2x).
    • Write the K expression.
    • Substitute the "E" row expressions into the K expression.
    • Solve for x. (Tip: Try the 5% approximation if K is small).
    • Check the approximation: (x / [Initial]) * 100% < 5%. If it fails, use the quadratic formula.
    • Plug the value of x back into the "E" row expressions to find the final equilibrium concentrations.
  5. Given: Kc and T. Find: Kp (or vice-versa).
    • Use the formula: Kp = Kc(RT)Δn.
    • Identify T in Kelvin (K = °C + 273.15).
    • Use R = 0.08206 L·atm/(mol·K).
    • Calculate Δn = (sum of moles of gas products) - (sum of moles of gas reactants).
    • Plug in all values and solve for the unknown K.
  6. Given: K and a set of initial concentrations. Find: Which direction the reaction will shift.
    • Write the K expression, but label it Q (Reaction Quotient).
    • Plug the given initial (non-equilibrium) concentrations into the Q expression.
    • Compare Q to K:
      • Q < K: Ratio is too small. Shifts RIGHT (→) to make products.
      • Q > K: Ratio is too large. Shifts LEFT (←) to make reactants.
      • Q = K: System is at equilibrium.
  7. Given: A system at equilibrium. Find: Which direction it will shift if you change T, P, or concentration.
    • Concentration: If you ADD a substance, shift AWAY. If you REMOVE, shift TOWARD.
    • Pressure (Gases): If you INCREASE P, shift to side with FEWER moles of gas. If you DECREASE P, shift to side with MORE moles of gas.
    • Temperature:
      1. Treat heat as a reactant (endothermic, ΔH > 0) or product (exothermic, ΔH < 0).
      2. If you ADD heat (increase T), shift AWAY from "Heat".
      3. If you REMOVE heat (decrease T), shift TOWARD "Heat".

Chapter 5: Acid-Base Definitions and Strength üß™

Defining acidity, quantifying it with pH, and calculating pH for strong and weak solutions.

Core Concepts

Acid-Base Definitions

  • Br√∏nsted-Lowry: Acid is a proton (H+) donor. Base is a proton acceptor.
  • Conjugate Pair: An acid/base pair that differs by one H+ (e.g., HF/F-).

Autoionization of Water (Kw)

H2O + H2O ⇌ H3O+ + OH-

Kw = [H3O+][OH-] = 1.0 x 10-14 (at 25¬∞C). This is true for ALL aqueous solutions.

The pH Scale

  • pH = -log[H3O+]
  • pOH = -log[OH-]
  • pH + pOH = 14.00
  • Acidic: pH < 7 | Neutral: pH = 7 | Basic: pH > 7

Strong vs. Weak

  • Strong Acids/Bases: Dissociate 100%. This is a stoichiometry problem. (e.g., HCl, NaOH).
  • Weak Acids/Bases: Dissociate partially. This is an equilibrium problem. (e.g., HF, NH3).

Dissociation Constants (Ka, Kb)

  • Ka (Acid): HA ⇌ H+ + A- ‚Üí Ka = [H+][A-] / [HA]
  • Kb (Base): B + H2O ⇌ BH+ + OH- ‚Üí Kb = [BH+][OH-] / [B]
  • Relationship: For a conjugate pair, Ka √ó Kb = Kw.

Salt Hydrolysis

Some ions react with water to change pH:

  • Neutral: Cation from Strong Base, Anion from Strong Acid (e.g., NaCl).
  • Basic: Anion is a conjugate base of a weak acid (e.g., NaF ‚Üí F- is basic).
  • Acidic: Cation is a conjugate acid of a weak base (e.g., NH4Cl ‚Üí NH4+ is acidic).
Key Skills & Problem Solving

Strong Acid/Base pH

This is stoichiometry. For 0.05 M HCl, [H3O+] = 0.05 M. For 0.05 M Ba(OH)2, [OH-] = 0.10 M (watch stoichiometry!)

Weak Acid/Base pH

This is an ICE table problem using Ka or Kb.

  1. Write the dissociation reaction.
  2. Set up ICE table to find [H3O+] or [OH-] (which is 'x').
  3. Solve Ka = x2 / ([Initial] - x). Use the 5% approximation.
  4. Check approximation.
  5. Calculate pH (or pOH, then pH).

Percent Ionization

Percent Ionization = ([H3O+]eq / [HA]initial) √ó 100%

Common Pitfalls
  • Confusing strong and weak. This is the #1 mistake. Don't use an ICE table for a strong acid!
  • Mixing pH and pOH. If you solve a weak BASE problem, 'x' = [OH-]. You must find pOH first, then pH.
  • Forgetting Ka √ó Kb = Kw. You are often given Ka for an acid but must find the pH of its conjugate base (a salt). You must calculate Kb first.
  • Assuming all salts are neutral (they are not!).
Expected Question Types
  1. Given: A reaction. Find: The Br√∏nsted-Lowry acid, base, conjugates.
  2. Given: One of pH, pOH, [H3O+], [OH-]. Find: The other three.
  3. Given: Concentration of a strong acid/base. Find: pH.
  4. Given: Concentration and Ka of a weak acid. Find: pH.
  5. Given: Concentration and Kb of a weak base. Find: pH.
  6. Given: Concentration and pH of a weak acid. Find: Ka (ICE table in reverse).
  7. Given: Ka for an acid. Find: Kb for its conjugate base.
  8. Given: Concentration of a salt (e.g., NaF, NH4Cl). Find: The pH (salt hydrolysis).

Chapter 5: Acid-Base Definitions and Strength üß™

Defining acidity, quantifying it with pH, and calculating pH for strong and weak solutions.

Core Concepts

Acid-Base Definitions

  • Br√∏nsted-Lowry: Acid is a proton (H+) donor. Base is a proton acceptor.
  • Conjugate Pair: An acid/base pair that differs by one H+ (e.g., HF/F-).

Autoionization of Water (Kw)

H2O + H2O ⇌ H3O+ + OH-

Kw = [H3O+][OH-] = 1.0 x 10-14 (at 25¬∞C). This is true for ALL aqueous solutions.

The pH Scale

  • pH = -log[H3O+]
  • pOH = -log[OH-]
  • pH + pOH = 14.00
  • Acidic: pH < 7 | Neutral: pH = 7 | Basic: pH > 7

Strong vs. Weak

  • Strong Acids/Bases: Dissociate 100%. This is a stoichiometry problem. (e.g., HCl, NaOH).
  • Weak Acids/Bases: Dissociate partially. This is an equilibrium problem. (e.g., HF, NH3).

Dissociation Constants (Ka, Kb)

  • Ka (Acid): HA ⇌ H+ + A- ‚Üí Ka = [H+][A-] / [HA]
  • Kb (Base): B + H2O ⇌ BH+ + OH- ‚Üí Kb = [BH+][OH-] / [B]
  • Relationship: For a conjugate pair, Ka √ó Kb = Kw.

Salt Hydrolysis

Some ions react with water to change pH:

  • Neutral: Cation from Strong Base, Anion from Strong Acid (e.g., NaCl).
  • Basic: Anion is a conjugate base of a weak acid (e.g., NaF ‚Üí F- is basic).
  • Acidic: Cation is a conjugate acid of a weak base (e.g., NH4Cl ‚Üí NH4+ is acidic).
Key Skills & Problem Solving

Strong Acid/Base pH

This is stoichiometry. For 0.05 M HCl, [H3O+] = 0.05 M. For 0.05 M Ba(OH)2, [OH-] = 0.10 M (watch stoichiometry!)

Weak Acid/Base pH

This is an ICE table problem using Ka or Kb.

  1. Write the dissociation reaction.
  2. Set up ICE table to find [H3O+] or [OH-] (which is 'x').
  3. Solve Ka = x2 / ([Initial] - x). Use the 5% approximation.
  4. Check approximation.
  5. Calculate pH (or pOH, then pH).

Percent Ionization

Percent Ionization = ([H3O+]eq / [HA]initial) √ó 100%

Common Pitfalls
  • Confusing strong and weak. This is the #1 mistake. Don't use an ICE table for a strong acid!
  • Mixing pH and pOH. If you solve a weak BASE problem, 'x' = [OH-]. You must find pOH first, then pH.
  • Forgetting Ka √ó Kb = Kw. You are often given Ka for an acid but must find the pH of its conjugate base (a salt). You must calculate Kb first.
  • Assuming all salts are neutral (they are not!).
Expected Question Types
  1. Given: A reaction. Find: The Br√∏nsted-Lowry acid, base, conjugates.
    • Identify the species that donates a H+ (this is the acid).
    • Identify the species that accepts the H+ (this is the base).
    • Find the acid's "partner" on the product side (what's left after donating H+) - this is the conjugate base.
    • Find the base's "partner" on the product side (what it becomes after accepting H+) - this is the conjugate acid.
  2. Given: One of pH, pOH, [H3O+], [OH-]. Find: The other three.
    • Start with your given value.
    • Use pH = -log[H3O+] or [H3O+] = 10-pH.
    • Use pOH = -log[OH-] or [OH-] = 10-pOH.
    • Use the master link: pH + pOH = 14.00.
    • Or use Kw = [H3O+][OH-] = 1.0 x 10-14.
  3. Given: Concentration of a strong acid/base. Find: pH.
    • Identify the acid/base as strong (this means 100% dissociation, no ICE table).
    • For a strong acid (e.g., 0.05 M HCl): [H3O+] = [Acid] = 0.05 M.
    • For a strong base (e.g., 0.05 M Ba(OH)2): Watch stoichiometry! [OH-] = 2 √ó [Base] = 0.10 M.
    • If you have [H3O+], calculate pH = -log[H3O+].
    • If you have [OH-], calculate pOH = -log[OH-], then pH = 14.00 - pOH.
  4. Given: Concentration and Ka of a weak acid. Find: pH.
    • Write the dissociation reaction: HA ⇌ H+ + A-.
    • Set up an ICE table.
    • Fill "E" row: [HA] = [Initial] - x, [H+] = x, [A-] = x.
    • Write the Ka expression: Ka = (x2) / ([Initial] - x).
    • Assume x is small (5% rule): Ka ≈ (x2) / [Initial].
    • Solve for x. x = [H3O+].
    • Check assumption: (x / [Initial]) * 100% < 5%.
    • Calculate pH = -log(x).
  5. Given: Concentration and Kb of a weak base. Find: pH.
    • Write the dissociation reaction: B + H2O ⇌ BH+ + OH-.
    • Set up an ICE table.
    • Fill "E" row: [B] = [Initial] - x, [BH+] = x, [OH-] = x.
    • Write the Kb expression: Kb = (x2) / ([Initial] - x).
    • Assume x is small (5% rule).
    • Solve for x. x = [OH-].
    • Check assumption.
    • Calculate pOH = -log(x).
    • Calculate pH = 14.00 - pOH.
  6. Given: Concentration and pH of a weak acid. Find: Ka.
    • This is an ICE table in reverse.
    • First, find x from the pH: x = [H3O+] = 10-pH.
    • Set up an ICE table.
    • Fill in the "E" row using the x you just found:
      • [H+] = x
      • [A-] = x
      • [HA] = [Initial] - x
    • Plug these equilibrium concentrations into the Ka expression: Ka = (x)(x) / ([Initial] - x).
    • Solve for Ka.
  7. Given: Ka for an acid. Find: Kb for its conjugate base.
    • Use the formula: Ka √ó Kb = Kw, where Kw = 1.0 x 10-14.
    • Rearrange: Kb = Kw / Ka.
  8. Given: Concentration of a salt (e.g., 0.1 M NaF). Find: The pH.
    • Identify the ions (e.g., Na+ and F-).
    • Determine if either ion is acidic or basic.
      • Na+ is the cation of a strong base (NaOH). It is neutral.
      • F- is the anion of a weak acid (HF). It is a weak base.
    • The solution pH is determined by the F-. This is now a weak base problem (see Question 5).
    • You will need Kb for F-. Find it using Kb = Kw / Ka (you'll be given Ka for HF).
    • Set up an ICE table for F- + H2O ⇌ HF + OH- and solve for x = [OH-], then find pOH and pH.
